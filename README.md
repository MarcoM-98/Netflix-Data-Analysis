 # How to run
 1. Clone the repo
 2. install the requirements such as pandas, matplotlib and seaborn in your python enviroment.
 3. open the notebook in jupyter lab and click on notebooks/analysis.ipynb
 4. reproduce the results by clicking on kernel and restart & run all
